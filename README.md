## 为什么是3Web Reserach

#### 3Web DAO是一个加密领域的自治理社区，DAO成员均有多年区块链加密领域的研究和投资背景,愿景是帮助独角兽落地和Web3投资者获利。

**3Web Analytics**：专注于区块链上的数据分析平台

**3Web Research**：专注于区块链加密领域的研究机构

**3Web Capital** : 专注于区块链加密领域的投资
